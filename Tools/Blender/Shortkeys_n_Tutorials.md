Blender [v3.6]

# 1. **Modelagem**

## **Atalhos Úteis:**
- **Tab**: Modo de edição
  - **Alt + Z**: Wireframe View
  - **A** Select All
  - **I** Insert Faces
  - **K**: Knife
    - **Z**: Knife -> Cut Through
  - **J**: Connect Vertex Path
  - **Ctrl + Mouse2**: Extrude to Cursor
  - **M**: Merge
  - **Alt + M**: Merge Menu
  - **G**: Grab/Move
  - **E**: Extrude
